## 5. State / Interaction Flow Summary

```
  User loads file               User hits Play
       в”Ӯ                              в”Ӯ
       в–ј                              в–ј
  <input type="file">         toggle_play(deck_state)
  вҶ’ FileReader                вҶ’ recreate AudioBufferSourceNode
  вҶ’ decodeAudioData()         вҶ’ .start(0, offset)
  вҶ’ extract waveform peaks    вҶ’ is_playing.set(true)
  вҶ’ update DeckState          вҶ’ platter begins spinning (rAF)
  вҶ’ waveform canvas redraws   вҶ’ waveform playhead scrolls (rAF)

  rAF loop (60 fps)
  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
  read current_time from AudioContext
  вҶ’ update current_secs signal
  вҶ’ check loop_out boundary вҶ’ re-schedule if looping
  вҶ’ draw_platter() on canvas
  вҶ’ draw_waveform() on canvas
  вҶ’ update VU meter signal (вҶ’ reactive DOM height update)
```
